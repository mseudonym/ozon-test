# Ozon test task

Тестовое задание в Ozon. Интерактивная карточка с круговым индикатором прогресса. Спроектирована как переиспользуемый
компонент с программным API.

## Структура проекта

```
progress-card/
├── index.html
├── styles/
│   ├── reset.css       # сброс браузерных стилей
│   └── main.css        # стили компонента
└── scripts/
    └── main.js         # класс ProgressCard + инициализация
```

## API

Карточка управляется через класс `ProgressCard`. Конструктор принимает любой DOM-элемент — это позволяет размещать
несколько независимых карточек на одной странице.

```js
const card = new ProgressCard(document.querySelector('.card'));
```

### Методы

| Метод               | Аргумент       | Описание                      |
|---------------------|----------------|-------------------------------|
| `setValue(val)`     | `number` 0–100 | Устанавливает размер дуги     |
| `setAnimated(bool)` | `boolean`      | Включает / выключает вращение |
| `setHidden(bool)`   | `boolean`      | Скрывает / показывает кольцо  |
| `getState()`        | —              | Возвращает текущее состояние  |

### Пример использования

```js
// установить прогресс
card.setValue(75);

// включить анимацию
card.setAnimated(true);

// скрыть круг
card.setHidden(true);

// получить текущее состояние
card.getState();
```

Вызов любого метода синхронизирует UI-контролы с состоянием — чекбоксы и инпут обновляются автоматически.

## Состояния

**Normal** — базовое состояние. `setValue(n)` управляет дугой: 0 — пустая, 100 — полная. Дуга начинается в 12 часов и
движется по часовой стрелке.

**Animated** — кольцо вращается по часовой стрелке с периодом 2 секунды. Независимо от Normal: анимацию можно включить
при любом значении Value.

**Hidden** — кольцо скрывается со страницы. Состояние Value и Animated при этом сохраняется.
